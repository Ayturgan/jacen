# Раздаточный материал: Карточки Персонажей

*Этот документ предназначен для игроков. Здесь собраны ваши стартовые данные, навыки и цели, какими вы их видите на начало пути.*

---

## 1. [[Эликс Тайэрис]] (Удачливый)
> *«Я не ищу величия. Я просто хочу, чтобы моя семья была в безопасности, а этот кулон перестал так странно греться».*

| **Категория** | **Ваши данные** |
| :--- | :--- |
| **Роль** | Искатель истины / Удачливый юноша |
| **Внешность** | Снежно-белые волосы, пронзительные изумрудно-зеленые глаза. Мягкие, дружелюбные черты лица. Одет просто, по-дорожному. |
| **Особая деталь** | Обсидиановый кулон на шее — ваша единственная связь с настоящим прошлым. |
| **Что знают другие** | Эликс знает, что он приемный. Семья Веллар нашла его после кораблекрушения. Он добр, порой наивен, но его «везение» с огнем уже стало легендой в узких кругах. |

## 2. [[Сайлас]] (Верный Щит)
> *«Мир полон фанатиков и убийц. Пока я жив, ни один из них не прикоснется к этому парню».*

| **Категория** | **Ваши данные** |
| :--- | :--- |
| **Роль** | Ветеран-телохранитель / Мастер выживания |
| **Внешность** | Суровый мужчина с жесткими черными волосами. Взгляд багровых глаз пугает многих. Лицо пересекает глубокий шрам над левой бровью. |
| **Особая деталь** | Массивный, идеально заточенный клинок без эмблем и украшений. |
| **Что знают другие** | Сайлас — тень Эликса. Он немногословен, практичен и всегда начеку. Его багровые глаза — результат «старой травмы» или редкой болезни, о которой он не любит говорить. |

## 3. [[Варо Антарион]] (Тень Волантиса)
> *«В Браавосе говорят, что у каждого секрета есть своя цена. Я пришел сюда, чтобы узнать цену своей свободы».*

| **Категория** | **Ваши данные** |
| :--- | :--- |
| **Роль** | Информационный брокер / Мастер маскировки |
| **Внешность** | Высокий, жилистый, с бледной кожей и тёмными волосами. Глаза цвета штормового серого неба. |
| **Особая деталь** | Тонкий белый шрам на запястье — след от «прошлых ошибок». |
| **Что знают другие** | Варо — человек, который знает всех и всё в Волантисе. Он двигается бесшумно, а его манеры выдают в нем того, кто привык общаться с высшим светом, хоть он и прячется в тенях. |

## 4. [[Лисандра зо Тал-Рахаз]] (Искательница Истины)
> *«Легенды — это просто истории, которые люди рассказывают, когда им страшно признать правду. Я верю только в то, что могу потрогать или расшифровать».*

| **Категория** | **Ваши данные** |
| :--- | :--- |
| **Роль** | Ученый-археолог / Дешифровщик |
| **Внешность** | Светлая кожа, карие глаза. Рыжевато-каштановые волосы собраны в косу. Руки часто в чернилах. |
| **Особая деталь** | Сложный медный прибор («Эфирный хронометр») на поясе. |
| **Что знают другие** | Лисандра — изгнанная аристократка, которая предпочла пыльные свитки бальным залам. Она видит мир как систему механизмов и текстов. Она ищет способ восстановить величие своего рода через знания прошлого. |
